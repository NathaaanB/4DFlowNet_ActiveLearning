# 4DFlowNet ActiveLearning

This is an implementation of Active Learning into the netowrk provided by Edward Ferdian 4DFlowNet. This project is done in the context of a Master Thesis at Karolinska Institutet in collaboration with Uppsala Universitet.

Master Thesis Student : Nathan Blanchard
Main KI supervisor : David Marlevi

The active learning model is given here with the uncertainty estimation given by the Deep Ensembles method. Example data are also given for small training.
